# 拼豆图纸生成器

一个基于图片上传、自动生成拼豆图纸，并要求卡密验证的示例网站。

## 功能说明

- 用户上传图片
- 用户输入卡密
- 后端验证卡密并激活/检查有效期
- 生成拼豆图纸并返回可预览的输出图片

## 卡密档位

- `DAY1-ABC123`：1天有效
- `DAY3-DEF456`：3天有效
- `MONTH1-GHI789`：30天有效

## 启动方式

1. 进入 `backend` 目录
2. 安装依赖：

```bash
pip install -r requirements.txt
```

3. 启动后台服务：

```bash
uvicorn main:app --reload
```

4. 打开 `frontend/index.html` 直接访问，或使用本地静态服务器

## 3D 拼豆功能落地可行性

### 结论

- 2D 图片生成拼豆图纸是可行的，核心工作为：图片缩放、颜色量化、网格化输出。
- 3D 拼豆则更复杂，需根据图片内容确定深度、结构与层级。

### 3D 拼豆落地方式

- 方案1：使用多张不同角度的输入图像重建体积模型
- 方案2：根据单张图片的明暗/边缘生成“假3D”分层拼豆结构
- 方案3：直接输入3D模型或点云，再转换为体素拼豆方案

### 风险与建议

- 如果只有单张图片，自动生成真实 3D 拼豆模型准确性有限。
- 更稳妥的是先实现 2D 生成功能，再根据需求增加“分层效果”或“高度投影”功能。
