# 拼豆图纸生成器 - 设计文档

## 📋 项目概览

一个基于图片上传、自动生成拼豆图纸，并要求卡密验证的专业在线工具。参考 beadpattern.net 竞品设计，融合现代化UI/UX。

## 🎯 核心功能

1. **图片上传与拖拽**
   - 支持JPG、PNG等常见格式
   - 拖拽上传或点击选择
   - 实时预览选择的图片

2. **卡密验证与权限管理**
   - 3档卡密：1天、3天、30天
   - 卡密激活后记录时间和有效期
   - 过期自动失效

3. **自动图纸生成**
   - Floyd-Steinberg颜色量化算法
   - 自动色号匹配（20种拼豆标准色）
   - 生成高清网格图纸

4. **结果展示与下载**
   - 图纸实时预览
   - 网格尺寸、色号数等统计信息
   - PNG直接下载
   - PDF下载功能（开发中）

## 📊 卡密管理

### 卡密格式
```json
{
  "code": "test",
  "duration_days": 30,
  "description": "1个月有效"
}
```

### 激活流程
1. 用户提交表单
2. 后端查询卡密有效性
3. 如未激活：记录激活时间和有效期
4. 如已激活：检查是否过期
5. 返回结果（成功/卡密无效/卡密已过期）

### 状态存储
```json
{
  "code": "test",
  "activated_at": "2026-05-23T10:30:45.123456",
  "duration_days": 30
}
```

## 🚀 部署指南

### 开发环境
```bash
# 后端
cd backend
python3 -m uvicorn main:app --reload --port 8001

# 前端（另一个终端）
cd frontend
python3 -m http.server 3000
```

访问 `http://localhost:3000`

### 生产环境建议
- 后端：Gunicorn + Nginx反向代理
- 前端：CDN分发，使用构建工具（Vite/Next.js）
- 数据库：替换JSON文件为PostgreSQL/MongoDB
- 文件存储：上传到云存储（AWS S3等）
- SSL/HTTPS：LetsEncrypt证书
